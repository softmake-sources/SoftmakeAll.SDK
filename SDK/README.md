SDK do Softmake All (pacote principal).

Este pacote contém algumas funcionalidades de apoio ao desenvolvimento de módulos para o Softmake All.
Principais funcionalidades implementadas:
- Consumo de serviços REST
- Compressão de arquivos em memória
- Escrita e Leitura de arquivos com tamanho de colunas delimitados
- Helpers para as bibliotecas, como System.Text.Json

Não deixe de consultar os pacotes adicionais.
